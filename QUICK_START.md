# 快速开始指南

欢迎加入义务维修队网站开发团队！本指南将帮助你在 5 分钟内开始开发。

## 🚀 5 分钟快速上手

### 第 1 步：Fork 仓库（首次）

1. 访问 https://github.com/Hyron-long/Friendly-Fix-It-Crew.github.io
2. 点击右上角的 **Fork** 按钮
3. 等待 Fork 完成

### 第 2 步：Clone 到本地

```bash
# 替换为你的用户名
git clone https://github.com/你的用户名/Friendly-Fix-It-Crew.github.io.git
cd Friendly-Fix-It-Crew.github.io
```

### 第 3 步：安装依赖

```bash
npm install
```

### 第 4 步：启动开发服务器

```bash
npm run dev
```

浏览器会自动打开 http://localhost:5173

### 第 5 步：开始修改

尝试修改 `src/views/Home.vue` 文件，保存后浏览器会自动刷新显示更改。
